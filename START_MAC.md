# Start ClaudeBot On macOS

1. Instala Python 3.11 o superior.
2. Clona o descarga este proyecto.
3. Abre Claude Desktop y dejalo visible.
4. Da permisos de `Accessibility` y `Screen Recording` a Terminal o iTerm.
5. Ejecuta:

```bash
chmod +x ./run_claudebot_mac.command
./run_claudebot_mac.command
```

Si en tu Mac Claude usa `Command+Enter`, entonces ejecuta:

```bash
./run_claudebot_mac.command --hotkey command,enter
```
