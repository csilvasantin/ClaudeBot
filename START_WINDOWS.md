# Start ClaudeBot On Windows

1. Instala Python 3.11 o superior.
2. Clona o descarga este proyecto.
3. Abre Claude Desktop y dejalo visible.
4. Haz doble clic en `run_claudebot.bat`. Ese lanzador lo arranca en segundo plano.

Si PowerShell pregunta por permisos de ejecucion, permite ejecutar el script.

Opcional:

```powershell
.\run_claudebot.ps1
```

Si quieres verlo en primer plano desde PowerShell:

```powershell
.\run_claudebot.ps1
```
