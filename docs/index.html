ClaudeBot

Deja que Claude siga trabajando sin vigilar el boton.

ClaudeBot observa la ventana de Claude Desktop, detecta la confirmacion visual para continuar, espera a que no estes moviendo el raton y lanza la hotkey correcta con evidencia before/after.

Ver en GitHub Arranque rapido

Que hace

Detecta el CTA de continuacion con OpenCV, enfoca Claude cuando toca y evita molestarte si estas usando activamente el raton.

Seguridad de uso

Antes de actuar, ClaudeBot espera 5 segundos de quietud del raton y vuelve a comprobar que la confirmacion sigue visible.

Evidencia visual

Guarda una captura antes y otra despues de cada accion para que puedas verificar exactamente lo que hizo.

Inicio rapido

Listo en un minuto

01

Windows

Clona el repo, abre Claude y haz doble clic en run_claudebot.bat.

02

macOS

Concede permisos de Accessibility y Screen Recording, luego ejecuta ./run_claudebot_mac.command.

03

Personaliza

Ajusta --mouse-idle-seconds, la region de captura o la hotkey si tu entorno lo necesita.
